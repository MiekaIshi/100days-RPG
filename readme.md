# プログラミング１００日チャレンジ
ここでは、「レトロRPGを１００日で実装していく」ことを目標にします。一種間分のタスクは、ジェミニ君に決めてもらいます。

## ファイル構成
* remdme.md このファイル:フォルダ内のファイル内容、ゲーム仕様、一週間ごとの作業予定の概要
* task.md 日々の作業予定と感想
* Main.java メインメソッドがあるクラス
* Player.java プレイヤー情報を持つクラス
* WordMap.java 地図に関するクラス:シングルトンパターンとして実装しました。こうすることで、一つのインスタンスを複数のクラスで共有できます。
* MapData.java WorldMapの構造体（マップ中の一つ一つのエリア）としてのクラス
* data-map.csv ワールドマップの地形データを二次元配列形で保持するファイル。
* Enemy.java 敵キャラの親クラス
* Boss.java ラスボスのクラス
* Slime.java みんな大好きスライムのクラス
* CraftManager.java クラフトクラス
* recipes.csv クラフトレシピ一覧
