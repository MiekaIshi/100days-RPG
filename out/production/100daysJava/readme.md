# プログラミング１００日チャレンジ
ここでは、「レトロRPGを１００日で実装していく」ことを目標にします。一種間分のタスクは、ジェミニ君に決めてもらいます。

## ファイル構成
* remdme.md このファイル:フォルダ内のファイル内容と、一週間ごとの作業予定の概要
* task.md 日々の作業予定と感想
* Main.java メインメソッドがあるクラス
* Player.java プレイヤー情報を持つクラス
* WordMap.java 地図に関するクラス:シングルトンパターンとして実装しました。こうすることで、一つのインスタンスを複数のクラスで共有できます。
* MapData.java WorldMapの構造体（マップ中の一つ一つのエリア）としてのクラス

## 一週間ごとの作業概要
* 1週間:この週では、マップの生成、プレイヤーの移動、そしてアイテムの配置というコアな仕組みをJavaで実装することに焦点を当てます。
* ２週間目：操作性の向上」と「マップ・アイテムの多様化」
